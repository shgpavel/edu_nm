Условие
Быстрое возведение в степень по модулю
Вычислите `(base^exp) mod mod` быстрым алгоритмом бинарного возведения в степень.

Формат входных данных
Три беззнаковых 32-битных числа: `base`, `exp`, `mod`.
Ограничения реального harness: `0 ≤ base, exp ≤ 2^32 - 1`, `1 ≤ mod ≤ 2^32 - 1`.
Формат выходных данных
Одно беззнаковое 32-битное число: `(base^exp) mod mod`.
Функция участника
typedef unsigned int u32;

u32 contest_pow_mod(u32 base, u32 exp, u32 mod);
Как вход и выход связаны с функцией
первое число из входа передаётся в `base`
второе число из входа передаётся в `exp`
третье число из входа передаётся в `mod`
возвращаемое значение функции печатается в `stdout`
свой `main()` писать не нужно
