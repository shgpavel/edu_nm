Условие
Demo Matrix Multiplication
Реализуйте умножение целочисленных матриц `A × B`.

Формат входных данных
В первой строке три числа: `m k n`.
Далее идут `m * k` целых чисел матрицы `A` в порядке row-major.
Затем идут `k * n` целых чисел матрицы `B` в порядке row-major.
Формат выходных данных
Выведите матрицу `C = A × B` размера `m × n`.
Результат печатается построчно.
Функция участника
void contest_matmul_i32(const int* a, const int* b, int* c, int rows_a, int cols_a, int cols_b);
Как вход и выход связаны с функцией
`rows_a = m`
`cols_a = k`
`cols_b = n`
`a` указывает на матрицу `A` размера `m * k` в row-major
`b` указывает на матрицу `B` размера `k * n` в row-major
`c` указывает на буфер результата размера `m * n` в row-major
после возврата из функции обёртка печатает содержимое `c` в `stdout`
свой `main()` писать не нужно
Контракт решения
Нужно определить функцию: void contest_matmul_i32(const int* a, const int* b, int* c, int rows_a, int cols_a, int cols_b);

A хранится как rows_a * cols_a, B как cols_a * cols_b, C как rows_a * cols_b. Все массивы row-major. Свои main(), clock() и код чтения теста писать не нужно. Для asm-реализации экспортируйте символ _contest_matmul_i32.
