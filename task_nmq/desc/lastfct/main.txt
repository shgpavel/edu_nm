Условие
Последняя ненулевая цифра факториала
Дано число `n`. Найдите последнюю ненулевую цифру числа `n!`.

Формат входных данных
Одно целое число `n`, где `0 ≤ n ≤ 100000`.
Формат выходных данных
Одна цифра от `1` до `9`: последняя ненулевая цифра `n!`.
Функция участника
int contest_last_nonzero_digit_factorial(int n);
Как вход и выход связаны с функцией
число из входа передаётся в аргумент `n`
функция должна вернуть последнюю ненулевую цифру факториала
возвращаемое значение печатается в `stdout`
свой `main()` писать не нужно
