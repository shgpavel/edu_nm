Условие
RC4 шифрование
Реализуйте алгоритм потокового шифрования `RC4`.

Формат входных данных
Первая строка: длина ключа `K`, где `1 ≤ K ≤ 256`.
Вторая строка: `K` целых чисел, байты ключа.
Третья строка: длина данных `N`, где `1 ≤ N ≤ 4096`.
Четвёртая строка: `N` целых чисел, байты открытого текста.
Формат выходных данных
Выведите `N` целых чисел: байты зашифрованного текста.
Функции участника
struct rc4_ctx {
    unsigned char S[256];
    int i;
    int j;
};

void contest_rc4_init(struct rc4_ctx *ctx, const unsigned char *key, int key_len);
void contest_rc4_crypt(struct rc4_ctx *ctx, const unsigned char *input, unsigned char *output, int len);
Как вход и выход связаны с функциями
обёртка создаёт структуру `struct rc4_ctx ctx`
байты ключа из входа передаются в `key`, а их количество в `key_len`
затем байты открытого текста передаются в `input`, а их количество в `len`
функция `contest_rc4_init` вызывается первой
после этого вызывается `contest_rc4_crypt`
массив `output` после `contest_rc4_crypt` печатается в `stdout`
свой `main()` писать не нужно
