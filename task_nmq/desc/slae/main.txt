Problem

Решение СЛАУ
Таймаут: 10 c, прогонов: 3, режим: function

Условие
Решение системы линейных алгебраических уравнений
Решите систему линейных уравнений `A x = b`.

Формат входных данных
Первая строка: `n` — размерность матрицы, `1 ≤ n ≤ 64`.
Следующие `n` строк: по `n` вещественных чисел — элементы матрицы `A`.
Последняя строка: `n` вещественных чисел — вектор правой части `b`.
Формат выходных данных
Если система имеет решение, выведите `n` вещественных чисел `x` с точностью `6` знаков после запятой.
Если система вырождена, обёртка печатает `SINGULAR`.
Функция участника
int contest_solve_linear_system(int n, double *A, double *b, double *x);
Как вход и выход связаны с функцией
`n` передаётся в аргумент `n`
`A` указывает на матрицу размера `n * n`, расположенную по строкам в row-major
`b` указывает на вектор правой части длины `n`
`x` указывает на выходной буфер длины `n`
функция должна вернуть `0`, если решение найдено, и ненулевое значение, если система вырождена
при возврате `0` обёртка печатает содержимое `x` в `stdout`
при возврате не `0` обёртка печатает `SINGULAR`
свой `main()` писать не нужно

Поддерживаются `.c`, `.cpp`, `.s`, `.S`, `.asm`. Посылка может состоять из одного или двух файлов. В этом режиме нельзя присылать свой `main`: раннер сам линкует ваше решение со своей обёрткой и вызывает функцию или набор функций, описанных в условии задачи.
